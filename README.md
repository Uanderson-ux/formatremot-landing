# FormatRemot - Landing Page

Esta é a página de vendas oficial do **FormatRemot**, uma ferramenta avançada para técnicos de TI realizarem formatações remotas com agilidade e precisão.

## Tecnologias
- HTML5 / CSS3
- JavaScript (Lucide Icons)
- Design Responsivo Premium

## Como visualizar
A página está configurada para ser hospedada via **GitHub Pages**.

## Notas sobre Assets
Os arquivos de mídia pesados (como o instalador .exe) são hospedados via Google Drive para garantir a melhor performance de download.
